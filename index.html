01
Mar 2026 Cloud / Azure NLP

Framing Lens — Asymmetric News Framing Detector

A cloud-native Azure application that detects per-entity linguistic framing bias in multilingual news statements. Integrates 8 Azure services including Azure OpenAI (GPT-4o mini), AI Language, Translator, Cosmos DB, and Cognitive Search to compute a Framing Asymmetry Score (FAS) and Cross-Lingual Framing Drift Score (CFDS) that quantifies how translation shifts perceived bias. Uses semantic role labeling to identify agent/patient dynamics and opinion mining on both original and translated text.

02
Apr 2026 Multi-Model DB

Evidence Custody Management System (ECMS)

A multi-model database architecture for managing digital forensic evidence across its full lifecycle. Integrates PostgreSQL (BCNF-normalized with PL/pgSQL triggers and stored procedures), Neo4j (custody chains as directed property graphs enabling O(k) traversal, cycle and gap detection via Cypher), and MongoDB (semi-structured audit logs with aggregation pipelines for insider threat detection). A Flask application layer coordinates writes across all three databases with role-based access control.

PostgreSQL Neo4j MongoDB Flask Supabase PL/pgSQL Cypher Python

Towards Safer Roads: End-to-End Traffic Violation Detection & License Plate Recognition

A computer vision system for real-time traffic violation detection in Indian road conditions. Uses YOLOv8 to detect violations including no helmet, no seatbelt, mobile phone use, and triple riding. Automated license plate recognition retrieves offender details from an RTO database and dispatches e-challans via SMS and email, removing the need for manual enforcement and reducing paper waste.
