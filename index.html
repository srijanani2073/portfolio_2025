Mar 2026 Cloud / Azure NLP

Framing Lens — Asymmetric News Framing Detector

A cloud-native Azure application that detects per-entity linguistic framing bias in multilingual news statements. Integrates 8 Azure services including Azure OpenAI (GPT-4o mini), AI Language, Translator, Cosmos DB, and Cognitive Search to compute a Framing Asymmetry Score (FAS) and a Cross-Lingual Framing Drift Score (CFDS) that quantifies how translation shifts perceived bias. Uses semantic role labeling to identify agent/patient dynamics and applies opinion mining on both original and translated text to measure framing drift across languages.

Apr 2026 Databases

Evidence Custody Management System (ECMS)

A multi-model database architecture for managing digital forensic evidence across its full lifecycle. Integrates three complementary DBMS paradigms: PostgreSQL (BCNF-normalized relational store with PL/pgSQL triggers and stored procedures) for structured evidence metadata and custody records; Neo4j for modeling custody chains as directed property graphs, enabling O(k) chain traversal, cycle detection, and gap analysis using Cypher; and MongoDB for semi-structured audit logs and behavioral profiling with aggregation pipelines for insider threat detection. Flask application layer coordinates writes across all three databases and enforces role-based access control.

PostgreSQL Neo4j MongoDB Flask Supabase PL/pgSQL Cypher Python
